# Achilles and Patroclus - Game Website

## 🎮 Complete Website Package

This website showcases your hand-drawn pixel art game with interactive animations, team credits, and donation support.

## 📁 Files Created

### Main Pages
- **MainWebsite.html** - Landing page with hero section, features, and level showcase
- **Gallery.html** - Interactive animation players for all Achilles & Patroclus animations
- **Credits.html** - Team member profiles and project stats
- **Donate.html** - Donation page with interactive slider
- **Style.CSS** - Complete styling with Greek mythology theme

## ✨ Features

### Gallery Page - Interactive Animation Players
All your hand-drawn animations are showcased with play/pause controls and speed sliders:

**Achilles Animations (6 total):**
- Front Walk (10 frames)
- Front Idle (5 frames)
- Back Walk (8 frames)
- Back Idle (8 frames)
- Side Walk (10 frames)
- Side Idle (6 frames)
- Side Attack (7 frames)

**Patroclus Animations (3 total):**
- Front Idle (8 frames)
- Healing (20 frames!)
- Side Walk (9 frames)

### Credits Page
- Team 2 member profiles: Cillian Lynch, Alan Haugh, Dylan Cooney, Jihan Xu
- Project statistics (100+ animation frames)
- Technology stack showcase
- Special thanks section

### Donation Page
- Interactive donation slider ($1-$100)
- Preset amount buttons ($3, $5, $10, $25)
- Dynamic tier descriptions based on amount
- Impact level breakdown

## 🎨 Design Features
- Greek mythology color scheme (gold #daa520 on dark navy)
- Smooth animations and transitions
- Responsive design for mobile, tablet, and desktop
- Pixel-perfect sprite rendering
- Floating animations and hover effects

## 🚀 How to View

1. Open **MainWebsite.html** in any web browser
2. Navigate using the top menu to explore all pages
3. On Gallery page, click "Play" buttons to see your animations in action!
4. Use speed sliders to control animation playback

## 💡 Next Steps for Live Deployment

To make this website live, you can:
1. **GitHub Pages** (Free) - Upload files to a GitHub repo and enable Pages
2. **Netlify** (Free) - Drag and drop the Cillian folder
3. **Vercel** (Free) - Connect your repo for automatic deployment
